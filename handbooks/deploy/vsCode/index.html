# 常用的插件

  1. Atom One Light Theme
    VSCode的主题。

  2. Code Runner 代码运行器
    运行多种语言的代码段或代码文件，和自定义命令。文档 (opens new window)

  3. CSS Compressor
    CSS压缩器，清洁和[紧凑| 缩小]您的CSS文件。
    实际上，您也可以使用选项美化您的CSS文件{format: 'beautify'}

  4. Debugger for Chrome
    VS代码-Chrome调试器

  5. ES7 React/Redux/GraphQL/React-Native snippets
    React语法高亮提示，格式化，Emmet，自动完成。

  6. ESLint
    javascript检测代码规范

  7. GitLens — Git supercharged
    内置到VScoder的Git的插件。它可以帮助您通过Git责任注释和代码镜头一目了然地看到代码作者的身份，无缝地导航和浏览Git存储库，通过强大的比较命令获得有价值的见解，等等。

  8. JavaScript (ES6) code snippets
    此扩展包含vscode编辑器的ES6语法的JavaScript代码片段（同时支持JavaScript和TypeScript）。
    官方文档 (opens new window)

  9. Jupyter
    Jupyter Notebook 是一个 IPython 交互式笔记本
    一些技巧 (opens new window)

  10. language-stylus
    stylus 语法高亮，选择器，属性，值，变量，函数等补全。

  11. Live Server
    启动具有实时重新加载功能的本地开发服务器，以处理静态和动态页面。

  12. Markdown All in One
    Markdown所需的全部功能（键盘快捷键，目录，自动预览等）。

  13. markdownlint
    Markdown检测代码规范。官方文档 (opens new window)

  14. Material Icon Theme
    vscode图标主题

  15. Python
    对Python语言的丰富支持（版本：2.7，> = 3.6），语法提示！

  16. REST Client
    REST Client允许您发送HTTP请求并直接在Visual Studio Code中查看响应。

  17. style-hook
    React中style样式语法提示。

  18. Vetur
    Vue语法高亮提示，格式化，Emmet，自动完成。
