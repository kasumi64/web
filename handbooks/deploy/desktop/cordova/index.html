# Cordova 应用流程

# 环境的搭建

  • 安装 Nodejs nodejs官网 (opens new window);
  • 安装 Cordova-cli 官网 (opens new window)
    1. npm install cordova -g;
    2. 添加 path 环境变量：C:\Users\admin\AppData\Roaming\npm\node_modules\cordova\bin
  • 安装 Java-jdk Java-jdk官网 (opens new window)
    1. Java SE 8 (1.8) Windows x64
  • 安装 Android Studio 官网 (opens new window)
    1. 安装 SDK，安卓的最小版本
    2. 安装 AVD 虚拟机
    3. 配环境变量，不修改默认路径可以不用配
      • SDK：ANDROID_HOME = ANDROID_SDK_ROOT = D:\Android sdk\SDK\
      • AVD：ANDROID_SDK_HOME = D:\Android sdk\AndroidAvd\
    4. Studio => file => settings... => System Setting => Android SDK => Android SDK Location 【D:\Android sdk\SDK\】
  • 安装 Ant 安卓平台下的打包附属工具官网 (opens new window)
    1. 左侧边栏 Download 下的 Binary Distributions
    2. 选择 apache-ant-1.10.9-bin.zip，然后解压把 bin 目录放到 path 环境变量中
  • Gradle build 时自动安装
    1. 网络不给力时手动安装
      • 官网 releases (opens new window)
      • 官网 services (opens new window)
    2. 安装路径 C:\Users\LGY\.gradle\wrapper\dists\【gradle-6.5-bin】
Cordova 架构图

# 创建工程项目

  1. cordova create xxx com.example.xxx HelloWorld
    • XXX 创建的目录
    • com.example.xxx 反向域名(包名)
    • HelloWorld 项目名称
  2. cd XXX
  3. cordova platform add android 添加平台
  4. cordova build android 编译app
  5. cordova platform remove android 删除平台
    cordova platform rm android 或者
  6. 运行app
    cordova emulate android
    cordova serve android
    cordova run android
  7. 引入插件，cordova9.0 安卓插件
    • cordova plugin add cordova-plugin-camera 相机
    • cordova plugin add cordova-plugin-device 设备
  8. 查看平台列表 cordova platform ls
  9. 查看插件列表 cordova plugin list
  10. cordova requirements 检查是否具备使用 Cordova 开发/运行 Android / IOS 应用的条件
- cordova plugin add cordova-plugin- .....
- cordova plugin add org.apache.cordova. .... //老式写法已淘汰

# 目录

  • platforms：已添加的打包平台，如：android，ios
  • plugins：引入的插件
  • www：H5开发目录

# 安卓工程目录

  • 打包好的文件 cordova工程\platforms\android\app\build\outputs\apk\debug\app-debug.apk
  • web文件 cordova工程\platforms\android\app\src\main\assets\www

# Cordova + vue 混合开发

修改 Vue 项目配置文件

  • 脚手架 3.xx 以上
// vue.config.js
module.exports = {
	outputDir:  path.resolve(__dirname, '../bulid_APK/www/'), //发布目录
	assetsDir: "static", //静态资源
	indexPath: "index.html" //入口文件
};
